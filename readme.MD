# Investment Analysis Project

## 简介
这是一个基于AKShare接口的 Python 的投资分析项目，旨在帮助用户进行股票数据分析、选股策略制定和自动化投资管理。该项目提供了多种功能模块，方便用户进行投资决策和管理。

## 功能
- **股票分析**：分析股票数据并计算相关指标。
- **选股策略**：根据用户自定义的策略筛选潜在投资股票。
- **API 接口**：提供股票数据的 API 接口，方便其他应用程序调用。
- **邮件通知**：定时发送投资相关的邮件通知。
- **成本平均法（DCA）**：支持使用成本平均法进行投资。
- **晚间交易**：实现晚间交易功能，适应不同的市场需求。
- **数据存档管理**：管理历史数据存档，方便后续分析。
- **配置管理**：使用配置文件管理项目参数。

## 技术栈
- **编程语言**：Python
- **配置管理**：INI 格式
- **邮件调度**：使用 Python 的邮件库
- **API 框架**：Flask 或 FastAPI（根据实际使用情况）
